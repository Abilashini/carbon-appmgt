Get API Documents

Get a list of documents belonging to an API.\n

URL

http://apis.wso2.com/api/appm/publisher/v1.0/apps/{appType}/id/{appId}/docs

HTTP Method

GET

Scope

appm:read

CURL Example

curl -H "Authorization: Bearer b0982cd2aacd463ff5f63cd5ebe58f4a" "http://127.0.0.1:9763/apps/webapp/id/8328392-d88d-dwu9-333r-r3"
