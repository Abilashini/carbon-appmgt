Delete Tags relevant to an App

Delete Tags relevant to an App by AppId.\n

URL

http://apis.wso2.com/api/appm/publisher/v1.0/apps/{appType}/id/{appId}/tags

HTTP Method

DELETE

Scope

appm:create

CURL Example

curl -X DELETE -H "Authorization: Bearer e099014668a524870c2895fe0595f125" -d '{"tags":["w2"]}' "http://localhost:9763/api/appm/publisher/v1.0/apps/webapp/id/80391fcb-45fc-4bb4-a83d-fc991a99084d/tags"

Sample Response
